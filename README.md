# WarSens — Warframe Sensitivity Advisor

딥러닝/AI 기반 워프레임 마우스 민감도 추천 프로그램.

사용자의 실제 플레이 데이터(마우스 입력 로그 + 화면 녹화 기반 조준 궤적 분석)와 마우스 사용 면적을 바탕으로, **개인 맞춤 최적 민감도**를 추천한다.

## 문서
- [기획서](기획서.md) — 프로젝트 목표, 아키텍처, 데이터 수집 전략, 로드맵

## 상태
현재 **Phase 0 — 기획 단계**. 프로토타입 착수 전.

## 기술 스택 (예정)
- Python 3.11+ / PyQt6
- dxcam (화면 캡처) + Win32 Raw Input (마우스 로거)
- YOLOv8 (적·크로스헤어 탐지)
- PyTorch / XGBoost (추천 모델)
