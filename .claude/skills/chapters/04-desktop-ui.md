# 데스크톱 UI 챕터

## 목적
Windows 데스크톱 앱으로 세션 관리 + 분석 대시보드 + 민감도 추천 제공.

## 프레임워크
- **PySide6** (LGPL, 상업 배포 용이) 권장
- 대안: PyQt6 (GPL — 배포 제약 있음)

## 주요 화면
1. **홈**: 현재 민감도/DPI 표시, "녹화 시작" 버튼
2. **녹화 중 오버레이**: 작은 상태 창 (녹화 중 / 프레임 카운트 / 정지 버튼)
3. **세션 분석 대시보드**:
   - 오버슈트율 타임시리즈
   - 평균 도달 시간
   - 마우스 이동 히트맵
4. **민감도 추천**: "현재 0.45 → 권장 0.40 (근거: 오버슈트율 22%)"
5. **설정**: DPI 입력, 패드 사이즈, 워프레임 실행 파일 경로

## 스레딩
- UI 스레드와 **녹화/분석 워커 스레드 분리 필수** (Qt의 `QThread`)
- 녹화 중 UI 프리즈 절대 금지

## 배포
- `pyinstaller --onefile` 로 exe 생성
- 사용자에게 "첫 실행 시 Windows 방어자가 경고할 수 있음" 안내
